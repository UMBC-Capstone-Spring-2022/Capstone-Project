MURA X-Ray Classification Using Deep Learning

Final Model & Evaluation
  • We will implement ensemble modeling for neural networks to improve our model performance and accuracy, then finalize our model based on the results
  • We will attempt to use Class Activation Mapping (CAM) to identify the discriminative regions that would influence the deep learning model to make a prediction
  • Our end product will be deployed using a Web API (user interface) and the user will be able to upload the X-ray images to identify abnormality or lack thereof
  • Since the study contains a collection of multi-view images from different angles, its final classification will be based on the threshold (0.6) on the average predicted probabilities
