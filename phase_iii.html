MURA X-Ray Classification Using Deep Learning

Model Tuning & Evaluation

To evaluate the accuracy of our model:

  • Base model accuracy was poor 58% with batch size 32, epochs = 2 with run time = 22 mins
  • We altered the batch size and epochs to analyze the model performance
  • Employing accuracy and loss function, we evaluated the ViT model and DenseNet Model
  • We applied resampling techniques to handle imbalanced datasets
  • Using a Pytorch sampler called ‘ImbalancedDatasetSampler’, which enabled us to rebalance the class distributions, we estimated the sampling weights, and mitigated overfitting by using data augmentation techniques
  • With this balanced dataset, we used the DenseNet model with batch size 64, epochs = 15 to save the best model with less loss
  • We attempted to ensemble neural networks, however, there is not much change with accuracy so to avoid high variance due to model complexity, we decided to use DenseNet/ViT as our final models based on their performance for different body parts
